# DBTI — Drinking Behavioral Type Indicator

**饮酒人格指标** — 一套基于五大维度的娱乐性饮酒人格测试。

30 道题，5 分钟，测出你在酒局上的真实人格。27 种核心人格 + 1 种隐藏人格，纯前端计算，不上传任何数据。

## 人格体系

### 五大维度

| 维度 | 含义 | H | M | L |
|------|------|---|---|---|
| **酒势 Drive** | 你和酒的主动性关系 | 猛 | 稳 | 避 |
| **酒能 Energy** | 醉后能量方向 | 燥 | 衡 | 静 |
| **酒魂 Soul** | 醉后情感深度 | 真 | 摇 | 铁 |
| **酒量 Capacity** | 物理耐受上限 | 海量 | 普通 | 一碰就倒 |
| **酒德 Grace** | 对周围人的影响 | 照顾全场 | 自得其乐 | 制造混乱 |

前三个维度（D × E × S）决定核心人格类型（3 × 3 × 3 = 27 种），后两个维度（C、G）作为修饰丰富描述。

## 计分机制

每题 3 个选项，分值为 `+3 / 0 / -3`。每维度 6 题，原始分范围 -18 ~ +18。

```
标准分 = (原始分 + 18) / 36 × 100

  0  ~ 33  → L
  34 ~ 66  → M
  67 ~ 100 → H
```

核心人格 = `D × E × S` 查表，修饰描述 = `C + G` 附加。

## 设计原则

- **无冒犯性** — 自嘲式幽默，没有贬义人格
- **多维度解释** — 不是贴标签，而是解释「你为什么会这样」
- **社交传播性** — 代号充满梗，适合截图分享
- **隐私友好** — 纯前端计算，零数据上传
